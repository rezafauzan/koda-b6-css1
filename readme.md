# Create responsive navbar using media query

navbar item appear horizontally on screen with width < 768px and appear vertically when screen has width more than 768px max 1024px

## On screen with width < 768px
![When screen size less then 768px its appear horizontally](/assets/img/Screenshot.png)

## On screen with width > 768px
![When screen size more then 768px its appear vertically](/assets/img/Screenshot2.png)

# How to run it
- Simply as download this repo source code
- open index.html in your browser
